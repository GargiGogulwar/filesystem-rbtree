# File System Simulator using C++ and Red-Black Tree



This project is an in-memory file system simulation implemented in C++.  

Each directory stores its subdirectories and files in a Red-Black Tree for fast insertion, search, and listing.



## Features

\- mkdir <name>

\- touch <name>

\- ls

\- cd <name>

\- cd ..

\- pwd

\- exit



## Compilation


g++ main.cpp filesystem.cpp rbtree.cpp -o fsim


./fsim
